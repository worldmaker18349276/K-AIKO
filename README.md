K-AIK▣  is a voice-controlled terminal-based rhythm game.

```
 ⣿⣴⣧⣰⣤⣄ [00000/00032] □   □⛶  □   ■       ■   □   □   ■   ■   □   [  0.9%|00:01]
```

## Installation

```
git clone git@github.com:worldmaker18349276/K-AIKO
cd K-AIKO
python -m pip install .
```

## Play

```
kaiko
```

Or play beatmap directly

```
kaiko <beatmap file path>
```
